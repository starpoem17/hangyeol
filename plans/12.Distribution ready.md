이 phase의 목적은 실제 검증만 제외한 배포 직전 상태를 만드는 것이다.
즉, 프로젝트 개발자가 직접 iPhone/Android에서 설치 테스트만 하면 되도록 준비하는 단계다.

주요 작업:

.env.example 및 env/secret 요구사항 정리
app-public 값과 server/function secret 구분 정리
Supabase migrations 배포 절차 정리
Supabase Edge Functions 배포 절차 정리
EAS build 설정 완료
internal distribution용 build profile 정리
app config에서 배포에 필요한 항목 정리
README 또는 루트 문서에 배포 절차 정리
“개발자가 실기기에서 무엇을 네가 직접 확인해야 하는지” 체크리스트 작성
    iOS 설치 확인
    Android 설치 확인
    로그인/온보딩
    고민 작성/답변
    푸시 수신 여부
TODO.md 배포 준비 항목 갱신

이 phase의 완료 기준:

우분투 환경에서 빌드/배포 준비 커맨드가 정리되어 있음
internal testing 직전까지의 설정 누락이 없음
이후 남는 것은 개발자가 직접 하는 실기기 검증뿐임
