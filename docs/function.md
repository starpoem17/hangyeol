# 주요 기능
## 기존 서비스들과 비교하여 갖는 차별화된 주요 기능
- 고민 게시자가 고민글을 올리는 경우 이는 공개 게시판에 개시되지 않는다. 공개 게시판 기능 자체가 존재하지 않는다. 대신 LLM을 통하여 익명의 맞춤 답변자에게 고민이 익명으로 전송된다. (게시자, 답변자 모두 익명으로 처리되며 애당초 개인정보를 수집하지 않는다.)
- 맞춤 사용자를 찾는 구체적 과정.
    - LLM의 입력으로 고민 게시자의 성별, 관심분야, 고민 내용이 들어간다.
    - LLM이 이를 바탕으로 적절한 고민 답변자 3명을 찾는다.
    - LLM은 각 고민 답변자들의 성별, 관심분야, 이전의 고민 게시 내용, 이전의 고민 답변 내용을 보고 적절한 고민 답변자를 찾는다. 현재는 프로토타입이고 사용자 수와 그 이용기간이 짧을 예정이기에 LLM이 모든 사용자의 모든 과거 기록을 보고 판단한다.
    - LLM이 최적 답변자를 찾지 못한 경우 차선 답변자라도 포함하여 어떻게든 3명을 찾도록 한다.
    - LLM이 답변자를 찾지 못했다고 fallback하는 경우는 허용하지 않는다.
- 고민 게시, 고민 답변 상황에서 LLM 필터링을 적용하여 부적절한 고민 게시 및 답변을 원천 차단한다. 구체적인 방안으로는 GPT api를 활용하여 LLM이 반사회적, 공격적, 혹은 다양한 이유로 고민 상담 플랫폼에 부적절하다고 여기는 경우 해당 답변의 전송을 원천 차단한다. 구체적인 차단 방식은 사용자가 글을 게시하려고 할 때 "부적절한 표현이 감지되었습니다."라는 경고 문구가 뜨고 글이 게시되지 않는 것으로 한다. 사용자는 해당 문구를 확인한 이후 "확인" 버튼을 눌러 경고 창을 닫을 수 있다. 이때 작성 중이던 글은 삭제되지 않은 상황이기에 그대로 이어쓰거나 수정하여 경고 문구를 회피할 수 있다.
- 고민 게시 상황에서 앱 첫 실행 시 수집한 정보를 바탕으로 해당 고민 답변에 적절한 사용자를 LLM이 찾고 해당 사용자에게 고민 답변을 부탁하는 알림이 전송된다.
- 고민 게시자는 고민 답변에 좋아요를 누를 수 있다.
- 고민 답변자는 고민 답변에 좋아요를 받는 경우 "고민을 해결해준 횟수" 카운터가 1 올라 성취감을 얻을 수 있다.

## 기타 주요 기능
- 앱 첫 실행 시 성별, 관심분야(연애, 가족, 학업, 취업 등)(원하는만큼 복수 선택 가능)를 선택
- 답변을 부탁하는 고민이 오면 알림
- 내 고민에 답변이 오면 알림
