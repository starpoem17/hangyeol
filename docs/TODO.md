# 본 프로젝트의 목적
1. 해당 프로젝트는 프로토타입으로써 Must-Have 구현만을 목표로 한다. 
2. 해당 서비스는 모바일(ios, android)로 배포된다.
3. 해당 서비스는 익명 고민 공유 플랫폼으로써 동작한다.

# Must-Have: 필수 핵심 기능 (이게 없으면 서비스가 안 됨)
- 고민을 작성하고 관심 분야 기반으로 적절한 고민 답변자에게 전송
- 앱 실행 시 첫 화면이 내 답변을 기다리는 고민들 목록
- 앱 첫 실행 → 관심 분야 선택
- LLM 필터링 기능을 통해 부적절한 고민 게시, 고민 답변을 차단한다.
- 고민에 대한 답변을 받은 후 좋아요 버튼 및 후기 작성

# Won't-Have
- 게임적인 기능 (랭킹 등)
- 초기 접속 → AI 대화를 통한 사용자 파악
- 이전의 고민들을 기록하는 보관소
- 앱 내에서 캡처방지기능
- 인증받은 사람들만 답변할 수 있도록하는 기능
